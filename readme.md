# advent of code 2023

https://adventofcode.com/2023

## setup

```
python3 -m venv .advent
source .advent/bin/activate
pip install -r requirements.txt
```

## activation

```
source .advent/bin/activate
```

## running

Unless otherwise noted, run these as
```
[perl|python|Rscript] script.ext < input.txt
```
